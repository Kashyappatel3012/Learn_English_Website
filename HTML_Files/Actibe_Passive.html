Active To Passive

Present Simple:

1. Positive:
→  Active: Subject + base form (v1) + object
→  Passive: Object + is/are + past participle + by + subject

2. Negative:
→  Active: Subject + do/does + not + base form (v1) + object
→  Passive: Object + is/are not + past participle + by + subject

3. Interrogative:
→  Active: Do/Does + subject + base form (v1) + object?
→  Passive: Is/Are + object + past participle + by + subject?

Past Simple:

1. Positive:
→  Active: Subject + past simple (v2) + object
→  Passive: Object + was/were + past participle + by + subject

2. Negative:
→  Active: Subject + did not + base form (v1) + object
→  Passive: Object + was/were not + past participle + by + subject

3. Interrogative:
→  Active: Did + subject + base form (v1) + object?
→  Passive: Was/Were + object + past participle + by + subject?

Future Simple:

1. Positive:
→  Active: Subject + will + base form (v1) + object
→  Passive: Object + will be + past participle + by + subject

2. Negative:
→  Active: Subject + will not + base form (v1) + object
→  Passive: Object + will not be + past participle + by + subject

3. Interrogative:
→  Active: Will + subject + base form (v1) + object?
→  Passive: Will + object + be + past participle + by + subject?

Present Continuous:

1. Positive:
→  Active: Subject + is/are + present participle (v-ing) + object
→  Passive: Object + is/are being + past participle + by + subject

2. Negative:
→  Active: Subject + is/are not + present participle (v-ing) + object
→  Passive: Object + is/are not being + past participle + by + subject

3. Interrogative:
→  Active: Is/Are + subject + present participle (v-ing) + object?
→  Passive: Is/Are + object + being + past participle + by + subject?

Present Perfect:

1. Positive:
→  Active: Subject + has/have + past participle + object
→  Passive: Object + has/have been + past participle + by + subject

2. Negative:
→  Active: Subject + has/have not + past participle + object
→  Passive: Object + has/have not been + past participle + by + subject

3. Interrogative:
→  Active: Has/Have + subject + past participle + object?
→  Passive: Has/Have + object + been + past participle + by + subject?

Past Perfect:

1. Positive:
→  Active: Subject + had + past participle + object
→  Passive: Object + had been + past participle + by + subject

2. Negative:
→  Active: Subject + had not + past participle + object
→  Passive: Object + had not been + past participle + by + subject

3. Interrogative:
→  Active: Had + subject + past participle + object?
→  Passive: Had + object + been + past participle + by + subject?

1. Positive:
→  Active: Subject + will have + past participle + object
→  Passive: Object + will have been + past participle + by + subject

2. Negative:
→  Active: Subject + will not have + past participle + object

3. Interrogative:
→  Active: Will + subject + have + past participle + object?

Modal Verbs:

1. Positive:
→  Active: Subject + modal verb (can, must, should, etc.) + base form (v1) + object
→  Passive: Object + can/must/should + be + past participle + by + subject

2. Negative:
→  Active: Subject + modal verb (can, must, should, etc.) + not + base form (v1) + object
→  Passive: Object + can/must/should + not be + past participle + by + subject

3. Interrogative:
→  Active: Modal verb (can, must, should, etc.) + subject + base form (v1) + object?
→  Passive: Can/Must/Should + object + be + past participle + by + subject?

1. Positive:
→  Active: Subject + to be + base form (v1) + object
→  Passive: Object + to be + past participle + by + subject

2. Negative:
→  Active: Subject + not + to be + base form (v1) + object

3. Interrogative:
→  Active: To be + subject + base form (v1) + object?
